# RealizaFesta

Site desenvolvido para uma empresa de kits personalizados para festas, com o objetivo de tornar ocasiões especiais em momentos únicos e memoráveis.

🔗 **Acesse o site:** https://flaviofrb.github.io/RealizaFesta/

---

## Motivação

Desenvolver esse site foi uma oportunidade de unir prática e aprendizado. Enquanto aprimorava meus estudos em desenvolvimento front-end, criei um projeto real com funcionalidades como filtros interativos, carrossel de produtos, zoom em imagens e integração com WhatsApp — recursos que exigiram aprofundamento em HTML, CSS, JavaScript e jQuery.

---

## Sobre o Projeto

A empresa oferece kits personalizados para festas, cuidando de cada detalhe para transformar celebrações em experiências memoráveis. O site foi desenvolvido para apresentar os produtos, facilitar o contato com a empresa e divulgar o trabalho nas redes sociais.

---

## Seções

- **Home** — banner rotativo com chamada para contato
- **Kits Pegue Monte** — galeria de kits com filtros por categoria (mais vendidos, personalizados, novos)
- **Topos de Bolo** — coleção de topos personalizados
- **Blogs** — registros de trabalhos realizados para clientes
- **Sobre** — apresentação da empresa e sua filosofia
- **Brindes** — catálogo de brindes divididos por categoria
- **Contato** — informações de contato, redes sociais e localização

---

## Funcionalidades

- Filtro de produtos por categoria com Isotope.js
- Carrossel de banners com Bootstrap
- Zoom em imagens de brindes ao clicar
- Botão de contato direto via WhatsApp
- Tooltips nas redes sociais
- Layout responsivo para mobile e desktop

---

## Tecnologias

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 5
- Isotope.js (filtro de produtos)
- Font Awesome (ícones)
